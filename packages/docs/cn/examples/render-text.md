# 文本渲染

在组件文件中，你可以添加一个 `script` 标签来编写组件的逻辑代码。通过暴露的 `default` 对象，可以设置组件实例化后的属性。

其中，`data` 属性用于定义组件元素的默认属性。一旦实例化完成，你就可以执行文本渲染等操作。
