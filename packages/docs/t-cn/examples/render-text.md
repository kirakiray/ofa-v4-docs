# 文本渲染

在組件文件中，你可以添加一個 `script` 標簽來編寫組件的邏輯代碼。通過暴露的 `default` 對象，可以設置組件參數。

其中，`data` 參數用於定義組件元素的默認數據。一旦實例化完成，你就可以使用**模板語法**。

文本渲染采用雙大括號進行包裹，即 `{{xxx}}`，其中的 xxx 表示組件的屬性名；
