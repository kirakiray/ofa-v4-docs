# 多级嵌套页面

要使用多级嵌套页面功能，只需在父层的嵌套页面模块上设置"parent"属性，并将其值设为要嵌套页面模块的地址。

更详细的信息，请参阅之前的文章，其中提供了与嵌套相关的参数和方法。
