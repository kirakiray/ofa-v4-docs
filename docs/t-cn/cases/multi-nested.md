# 多級嵌套頁面

要使用多層嵌套頁面功能，只需在父層的嵌套頁面模塊上設置"parent"屬性，並將其值設為要嵌套頁面模塊的地址。

更詳細的信息，請參閱之前的文章，其中提供了與嵌套相關的參數和方法。
