# 多重な入れ子のページ

親ページによる複数のネストされたページ機能を使用する場合、親レベルのネストされたページモジュールに「parent」属性を設定し、その値をネストされたページモジュールのアドレスに設定するだけです。

前の記事を参照してください。ネストに関連するパラメータとメソッドが提供されています。
