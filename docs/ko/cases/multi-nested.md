# 다단계 중첩 페이지

다층 중첩 페이지 기능을 사용하려면, 부모 계층의 중첩 페이지 모듈에 "parent" 속성을 설정하고 중첩 페이지 모듈의 주소를 값으로 설정하면 됩니다.

자세한 내용은 중첩과 관련된 매개변수 및 방법을 제공하는 이전 문서를 참조하세요.
