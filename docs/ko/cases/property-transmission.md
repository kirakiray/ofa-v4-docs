# 속성 전달

속성 전달을 사용하여 템플릿 문법으로 컴포넌트의 데이터를 템플릿 내의 요소 속성에 적용할 수 있습니다.

사용 방법은 속성에 `:name="key"`를 추가하는 것입니다. 여기서 `name`은 설정할 대상 요소의 속성 이름이며 `key`는 사용자 정의 컴포넌트에서 전달할 속성 키 이름입니다.
